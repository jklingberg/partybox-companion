# partybox-companion

Turn any PartyBox into a true WiFi speaker — the full appliance package.

Bundles the [`partyboxd`](../partyboxd) daemon with a CLI client (`partybox`) and
companion services. Run `partybox-companion` to start the appliance, or `partybox`
to talk to a running daemon over HTTP.

Part of the [partybox-companion](https://github.com/jklingberg/partybox-companion) workspace.
