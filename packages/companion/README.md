# partybox-companion

Turn any PartyBox into a true WiFi speaker — the full appliance package.

Bundles the [`partyboxd`](../partyboxd) daemon with the Companion Portal and
service orchestration (Spotify Connect, Bluetooth audio, WiFi provisioning).
Run `partybox-companion` to start the appliance; manage it from the Portal at
`http://partybox.local` or the REST API.

Part of the [partybox-companion](https://github.com/jklingberg/partybox-companion) workspace.
